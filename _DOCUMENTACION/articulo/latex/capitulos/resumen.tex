\begin{abstract}
    Este trabajo investiga las posibilidades que pueden ofrecer los enfoques de arquitectura dirigida por modelos (MDA) en el desarrollo y generación de código asistidos por IA, centrándonos en un ámbito de programación muy concreto: el diseño y generación de sintetizadores de sonido software. Actualmente, la falta de flujos estructurados para este tipo de desarrollos y las limitaciones del modelado UML —poco adaptado a conceptos musicales y excesivamente verboso cuando se estereotipa para ello— dificultan la transición del análisis a la implementación en este tipo de software. Para resolverlo, se proponen lenguajes de dominio específico (DSL) para modelos CIM y PIM, junto con el uso de la herramienta \textit{MDAA}, desarrollada para este trabajo, con el fin de facilitar tanto la fase de diseño como de generación de infromación procesable por la IA generadora de este tipo de software, y con miras a poder ser utilizada de forma integral como entorno de análisis/diseño/desarrollo manteniendo la correlación entre los niveles de abstracción MDA. Esta propuesta mejora la trazabilidad, estandariza el desarrollo de software de audio y permite que perfiles no ingenieriles gestionen el desarrollo de sistemas complejos de síntesis de sonido mediante modelos de abstracción, con toda la mejora que esto puede conllevar de cara a la reutilización y el refinamiento de los productos generados.
\end{abstract}
