# Electronic Ambient Synthesizer

Arquitectura de síntesis modular avanzada para paisajes sonoros etéreos y evolutivos. El sistema utiliza un enfoque multicanal para drones armónicos, texturas de ruido orgánico y filtrado dinámico, orquestado por un motor de modulación espacial.

Bloques clave:
- **Drone Core**: 3 osciladores con micro-desafinación.
- **Texture Gen**: Ruido rosa y browniano granular.
- **Spectral Filter**: Filtros Ladder y ecualización.
- **FX Master**: Reverb de 8s y delay atmosférico.

Diseño basado en el paradigma MDA con modelos CIM y PIM validados para una producción profesional.
